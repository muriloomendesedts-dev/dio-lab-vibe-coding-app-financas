# 💸 App de Organização de Finanças Murilo Mendes



```txt
Documento de Requisitos do Produto (PRD)

1. Contexto
Aplicativo de Organização de Finanças Pessoais baseado em interação por conversa.
Objetivo: simplificar o controle financeiro, eliminando formulários extensos ou planilhas complexas,
com uma experiência natural e intuitiva.

2. Problema
Usuários abandonam apps de finanças porque:
- Exigem entrada manual excessiva
- Possuem baixa personalização
- Não oferecem orientação prática para economia

Solução: experiência conversacional + recomendações automáticas para incentivar hábitos financeiros saudáveis.

3. Público-Alvo
- Iniciantes no controle financeiro
- Usuários que buscam praticidade e simplicidade
- Pessoas que desejam aprender a organizar suas finanças sem complexidade técnica

4. Objetivos do Produto
- Tornar o controle financeiro acessível e menos burocrático
- Oferecer insights personalizados de economia
- Criar experiência educativa e engajante com “Agente Financeiro” virtual

5. Funcionalidades-Chave
1. Registro de gastos via chat em linguagem natural
2. Classificação automática das transações
3. Definição e acompanhamento de metas financeiras
4. Dicas de economia personalizadas do “Agente Financeiro”
5. Relatórios simples e visuais com insights sobre hábitos de consumo

6. Requisitos Não Funcionais
- Usabilidade: interface intuitiva, linguagem acessível e tom educativo
- Segurança: proteção de dados financeiros e pessoais
- Escalabilidade: suporte a diferentes volumes de usuários e transações
- Compatibilidade: funcionamento em dispositivos móveis (Android/iOS)

7. MVP (Produto Mínimo Viável)
Telas Principais:
- Conversa: registrar gastos e metas
- Relatórios: visão geral dos gastos e progresso
- Dicas: recomendações do Agente Financeiro

Recursos Necessários:
- Processamento de linguagem natural (NLP)
- Algoritmo de categorização automática
- Módulo de metas financeiras
- Sistema de relatórios simples e visuais

Validação Inicial:
- Testes com grupo piloto de iniciantes
- Avaliação de facilidade de uso e clareza das recomendações
- Feedback sobre engajamento e utilidade das dicas

8. Critérios de Sucesso
- Registro de gastos em menos de 30 segundos
- 70% dos gastos classificados automaticamente com precisão
- 60% dos usuários relatam maior clareza sobre hábitos financeiros após 1 mês
- Retenção superior a 50% após 3 meses
```
